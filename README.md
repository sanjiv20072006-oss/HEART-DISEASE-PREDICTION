**Heart Disease Prediction using Machine Learning** is a healthcare-based web application that predicts the likelihood of heart disease using patient medical data. The system analyzes inputs such as age, gender, blood pressure, cholesterol level, heart rate, chest pain type, and other health parameters to determine whether a person is at risk of heart disease.

The project uses a machine learning model (Random Forest or Transformer, depending on the implementation) trained on a heart disease dataset. Users can enter patient details through a simple web interface, and the model instantly generates a prediction along with the estimated risk.

This project demonstrates data preprocessing, machine learning model training, prediction, and web application deployment using Python, Flask, HTML, and CSS. It is intended for educational purposes and showcases how artificial intelligence can support early disease risk assessment.
